# Step to run project :-

1) After downloading project from github, go to lost-votes folder.
2) run ng build --watch command for localy build.
3) or also use ng build --prod for production.

Note:- please install angular-cli environment and other node modules to run project locally.

# usage of this application :-

1) Any voter of India give vote from any country.
2) There will be no any vote lost. 
3) Data is fully secured and encrypted.
